# Prompt — Final Analytical Report

## Copy-paste prompt

You are a senior technical writer and AI systems reviewer.

Write a final analytical report for **Aircraft Maintenance Decision Copilot**.

## Required sections

1. Executive summary.
2. Current status.
3. Problem statement.
4. Proposed solution.
5. User and use cases.
6. Product requirements.
7. C1–C4 maturity architecture.
8. High-level architecture.
9. Data ingestion flow.
10. Data models.
11. Agentic AI architecture:
    - LLM
    - harness
    - tools
    - memory
    - context
    - sub-agents
    - agent loop
12. Retrieval strategy.
13. Security and governance.
14. Evaluation methodology.
15. Observability:
    - LangSmith or Langfuse
    - Prometheus and Grafana
16. Cloud deployment.
17. Demo results.
18. Business model.
19. Limitations.
20. Future work.
21. Conclusion.
22. References.

## Requirements

- Distinguish implemented, partially implemented, and proposed features.
- Include diagrams where helpful.
- Include measured metrics only.
- Explain design trade-offs.
- State that the system is decision support and requires human approval.
- Include reproducibility instructions.
- Include an appendix with API, schema, and evaluation details.

## Required output format

Produce `docs/final-report.md` with a professional, evidence-based tone. Add `TBD` markers for missing measurements instead of inventing values.
