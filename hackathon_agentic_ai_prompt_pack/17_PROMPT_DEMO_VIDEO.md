# Prompt — Demo Video Script and Recording Plan

## Copy-paste prompt

You are a hackathon demo director.

Create a concise, high-impact demo video plan for **Aircraft Maintenance Decision Copilot**.

## Target duration

2–4 minutes.

## Required story

1. One-sentence problem.
2. Why current manual cross-referencing is difficult.
3. Enter a defect case.
4. Show the agent workflow.
5. Show retrieved evidence and citations.
6. Show missing information or conflict detection.
7. Show the structured recommendation.
8. Show human approval.
9. Show agent trace or evaluation.
10. Show architecture and enterprise deployment path.
11. End with value proposition and future vision.

## Requirements

- Use one stable seeded case.
- Avoid typing during the recording when possible.
- Preload data.
- Hide API keys and personal information.
- Record a backup version.
- Keep all claims aligned with the working prototype.
- Do not imply autonomous aircraft release.
- Show failure handling briefly if stable.

## Deliverables

Create:

```text
docs/demo-script.md
docs/demo-shot-list.md
docs/demo-checklist.md
```

The script must include:

- Timestamp.
- Screen action.
- Voice-over.
- Expected system result.
- Backup action if the feature fails.

Also create a 30-second fallback demo script for live judging.
